NAO basics

Sun, Feb 5, 2017 NAO

In this document a set of notes and useful links about NAO robot are presented.

Nao safety

  • Best practices: http://doc.aldebaran.com/1-14/nao/practices.html
  • Put the robot in a stable position before “releasing stiffness”

Battery

  • From 30 minutes to 2 hours.
  • Charging the battery: http://doc.aldebaran.com/1-14/nao/battery-charging.html

Turn on the robot

  • Press the chest button once.
  • The boot process is completed when NAO says “OGNAK GNOUK” while its Chest button becomes white. < http://doc.aldebaran.com/1-14/nao/nao-turn-on.html>

Know IP address

  • Press the chest button once, when it’s on

Battery level

  • Indicated in the chest: green, yellow and blinking red.

Release stiffness

  • Press two times chest button.

Shutdown the robot

  • Press the chest button for 3 seconds, until NAO says “GNUK GNUK”.
  • The shutdown process takes approximately 18 seconds
  • The shutdown process is completed when all the LEDs are off.

NAO configuration

Nao wifi connection: http://doc.aldebaran.com/1-14/nao/nao-connecting.html

Problems with NAO?

  • Press the chest button for 8 seconds.

NAO basic software:

NAO basic softwares are: - Choregraphe: for create animation, dialogs and behavior. Also for simple simulation.

http://doc.aldebaran.com/1-14/software/choregraphe/index.html#choregraphe

  • Monitor: Feedback of the state and sensors of the robot.
  • SDK: Can be used to remotely control, create new NAOqi modules and enrich Choregraphe box library (use python)
  • OpenNAO: Operating System of the robot.

http://doc.aldebaran.com/1-14/dev/tools/opennao.html#opennao-advanced

  • NAOqi: is the main software that runs on the robot.

http://doc.aldebaran.com/1-14/dev/naoqi/index.html#naoqi-framework-overview
