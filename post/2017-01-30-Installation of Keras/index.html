Installation of Machine Learning libraries for my master thesis

Mon, Jan 30, 2017 deep learning

This post contains the steps followed to perform the installation of the machine learning libraries used for my thesis.

Python Installation

The python version used is 2.7 of 32-bit from the Anacoda distribution: https://www.continuum.io/downloads

Update pip install

python -m pip install --upgrade pip

Installation of machine learning libraries

# Used in MAC OS to solve the error: "HDF5 library version mismatched error"
conda install h5py

# Theano library
pip install theano

## TensorFlow library: Used only for MAC OS X and Linux
pip install tensorflow

# Keras library (In windows opening the CMD as administrator)
pip install keras
conda install mingw libpython

Keras backend

Keras has two backend implementations (2017): TensorFlow and Theano. The default backend implementation is TensorFlow.

In Windows there is only the Python 3.5 version of TensorFlow. In order to use Keras + Python 2.7 in Windows, the next steps can be performed:

  • Go to C:\Users\<user name> and open the folder .keras
  • Open the file keras.json and change the lines as:
{
    "image_dim_ordering": "th",
    "epsilon": 1e-07,
    "floatx": "float32",
    "backend": "theano"
}

More info in: https://keras.io/backend/
