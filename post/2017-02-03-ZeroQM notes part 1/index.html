ZeroMQ notes 1

Fri, Feb 3, 2017 ZeroMQ

This post present a set of notes and python examples of ZeroMQ.

Instalation:

First intall the libzmq libraries:

http://zeromq.org/intro:get-the-software

The install the binaries in Python using the next line:

# (Windows or OS X)
pip install --wheel pyzmq
# or
easy_install pyzmq
# or (pretty much anywhere)
pip install pyzmq

More info in: http://zeromq.org/bindings:python

ZeroMQ main idea

ZeroMQ is en general a middleware that augments the capabilities of TCP/IP sockets. Is a socket system that allow to do concurrency.

  • Is fast
  • Use to connect several parts of a distributed system
  • Works asynchronously (unlike regular socket implementation that works synchronously)
  • Allows the implementation of several comunication patterns as request/reply and publish/subcriber

Context in ZeroMQ

Used to help to manage the sockets created in ZeroMQ. They are created in the start of the aplication.

Request/Reply example

For this we will use the type sockets zmq.REP for the server and zmq.REQ for the client.

Server code

First import the zmq libraries

import zmq
import time

then, define a new context

context = zmq.Context()

now define the socket for Request/Reply patter using zmq.REP as argument on the context

sock = context.socket(zmq.REP)
sock.bind("tcp://127.0.0.1:5685")

run the server, get the client message and send a response “Echo” + message

while True:
    message = sock.recv()
    sock.send("Echo: " +  message)
    print ("Echo: " +  message)

Putting all together:

import zmq
import time
context = zmq.Context()
sock = context.socket(zmq.REP)
sock.bind("tcp://127.0.0.1:5685")
while True:
    message = sock.recv()
    sock.send("Echo: " +  message)
    print ("Echo: " +  message)

To exit press ctrl+x (Termianal in OSX/Linux)

Client code

The client code it is similar but we use the next elements instead:

  • zmq.REQ
  • socket.connect
import zmq
import time
context = zmq.Context()

# Connect with the server
sock = context.socket(zmq.REQ)
sock.connect("tcp://127.0.0.1:5685")

# Send a message using the socket

sock.send("hello")
print (sock.recv())
