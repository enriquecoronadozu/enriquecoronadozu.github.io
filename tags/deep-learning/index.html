Installation of Machine Learning libraries for my master thesis

Description This post contains the steps followed to perform the installation of the machine learning libraries used for my thesis. Python Installation The python version used is 2.7 of 32-bit from the Anacoda distribution: https://www.continuum.io/downloads Update pip install python -m pip install –upgrade pip Installation of machine learning libraries # Used in MAC OS to solve the error: "HDF5 library version mismatched error" conda install h5py # Theano library pip install theano ## TensorFlow library: Used only for MAC OS X and Linux pip install tensorflow # Keras library (In windows opening the CMD as administrator) pip install keras conda install mingw libpython Keras backend Keras has two backend implementations (2017): TensorFlow and Theano.
